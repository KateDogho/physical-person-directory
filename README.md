# Physical Person Directory

## Run Application

Run SQL Server.

Create Database using script:

```sql
CREATE DATABASE PhysicalPersonDirectory;
```

Replace connection string in [appsettings.Development.json](https://github.com/KateDogho/physical-person-directory/blob/main/PhysicalPersonDirectory.Api/appsettings.Development.json)

Build App.

Run App.

Seed data will be created during the first run
